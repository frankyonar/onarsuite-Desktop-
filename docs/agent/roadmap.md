# Roadmap Max Desktop

## Completato nell'MVP locale

- Electron + React + TypeScript.
- Pairing client, heartbeat e stato connessione.
- Workspace e cartelle autorizzate.
- Parsing PDF, DOCX, XLSX, CSV, TXT e MD.
- Upload e azioni bozza con conferma.
- Audit, coda offline, retry e idempotenza.
- Test della policy percorsi e build tipizzata.

## Prossima fase obbligatoria

1. Implementare OnarSuite Agent Gateway in Laravel 12.
2. Aggiungere migrazioni, policy tenant, token device e test API.
3. Creare UI OnarSuite per device, permessi, eventi e revoca.
4. Eseguire il flusso end-to-end pairing, heartbeat, upload, audit e revoca.

## Fasi successive

- Polling comandi e conferme locali.
- Packaging firmato Windows/macOS e aggiornamenti automatici.
- Antivirus/content scanning server-side.
- API Max per analisi e generazione bozze.
- OnarSuite MCP Server, automazione browser e app locali solo dopo revisione sicurezza dedicata.
